# Glossary

A glossary of common terms used in this project.

```{glossary}
CSV
    "Comma-separated values." A spreadsheet format stored as text, in which
    values of different columns are separated by commas.

Shortcuts
    A no-code automation development framework that works on iOS, iPadOS,
    iWatchOS, and macOS. It allows users to create workflows that perform
    actions. The actions can include using the camera, recognizing text in
    images, creating files, speaking text and recognizing voice commands,
    if-then logic, repeat loops, and more.

```
